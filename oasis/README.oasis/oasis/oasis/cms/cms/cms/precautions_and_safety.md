SOC Safety and Precautions Extraction (CMS-aligned)

The agent must explicitly address each item below (extract it, or state "not provided"):

Critical
- Weight-bearing status per limb and any orthopedic precautions
- Oxygen use (none vs in use); if in use: device type, ordered flow settings, and any activity instructions

Additional
- Cardiac/medical activity restrictions (if documented)
- Fall risk precautions and supervision needs (if documented)
- Cognitive/behavioral safety risks affecting mobility (if documented)
- Equipment affecting mobility/safety (IV, catheter, wound vac, drains) if documented

Rules
- Do not guess.
- Do not use vague phrases like "precautions observed" without listing the actual precautions.
